AgriSmart Hub Website  
---------------------  

AgriSmart Hub is a modern, responsive website designed to showcase intelligent agricultural solutions, services, and innovations. The site is built using HTML, CSS, and Bootstrap 5, and is organized for easy customization and extension.

Folder Structure  
----------------  
about.html              - About Us page with mission, vision, and sustainability goals  
contact.html            - Contact page with a form, company details, and map  
index.html              - Homepage with hero section, solutions grid, and CTA  
innovations.html        - Innovations page featuring carousel, video, and statistics  
services.html           - Services page listing premium agricultural services  
style.css               - Main stylesheet for all pages  
images/                 - Folder for all image assets (see below for required filenames)  

How to Use  
----------  
1. Place all HTML files and style.css in the root folder.  
2. Add the required images to the images/ folder. See the list below for filenames.  
3. Open index.html in your browser to view the homepage.  
4. All navigation links work between pages.  
5. The site uses Bootstrap 5 via CDN for layout and components.

Customization  
-------------  
- Edit style.css to change colors, fonts, or layout.  
- Update HTML files to modify content, add new sections, or change navigation.  
- Add a favicon.ico for a custom browser tab icon (optional).
